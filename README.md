## Descripción
Este repositorio contiene la entrega de la **Fase 1** y la **Fase 2** del proyecto de la materia **Introducción a la Inteligencia Artificial**.

En la carpeta `myquestions` se incluyen cuatro preguntas propuestas y sus respectivos generadores de casos de uso aleatorios, siguiendo la estructura solicitada en la guía del proyecto.

En la carpeta `myanswers` se incluyen las soluciones a las cuatro preguntas asignadas de compañeros, junto con el archivo de diagnóstico.

## Notas
Cada archivo `question-000X.txt` contiene el enunciado completo de una pregunta.  
Cada archivo `question-000X-usecase-generator.py` contiene la función generadora de casos de uso correspondiente.  
Cada archivo `answer-XXXX.py` contiene la solución a la pregunta con ese identificador, asignada en la Fase 2.  
El archivo `diagnostico.txt` documenta observaciones sobre las soluciones cuya validación automática puede verse afectada por errores en los generadores de casos de uso.
